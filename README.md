# Breakout - Rust/macroquad with Perlin Noise background

A performance-oriented implementation of the classic "Breakout" written in Rust using the Macroquad graphics library. This project features a custom-built 2D Perlin noise generator for dynamic, procedural background rendering.

## OVERVIEW

This project demonstrates the integration of procedural content generation within a real-time game loop. The background is not a static asset but a mathematically generated texture produced by a hand-rolled Perlin noise implementation. The game includes a fully functional UI system for state management, settings persistence, and score tracking.
